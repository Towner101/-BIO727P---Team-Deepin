Cluster Analysis Details

The cluster analysis was performed using the Python software package. The analysis used the following parameters:

  • Number of clusters: 5
  • Distance metric: Euclidean
  • Linkage criterion: Ward

The analysis identified 5 distinct clusters of samples based on their genetic similarity. The cluster plot shows the samples as points in a two-dimensional space, where the x-axis and y-axis represent the first two principal components of the genetic data. The samples are colored according to their assigned cluster.
